# Lista de Exercícios: Desenvolvimento Web com Flask e HTML5 🚀

Este repositório contém uma série de exercícios práticos desenvolvidos para a disciplina de Desenvolvimento Web 2. O objetivo é explorar os conceitos fundamentais do framework **Flask**, integração com **HTML5** e estilização rápida com **Pico.css**.

## 🛠️ Tecnologias Utilizadas

* **Python 3.x**
* **Flask** (Micro-framework web)
* **Jinja2** (Engine de templates)
* **Pico.css** (Framework CSS minimalista via CDN)
* **HTML5**

## 📂 Estrutura do Projeto

O projeto está organizado em pastas separadas para cada exercício, mantendo um ambiente virtual (`venv`) compartilhado na raiz:

```text
WEB2/
├── exercício_1/      # Conversor de Temperatura (°C para °F)
├── exercício_2/      # Calculadora de Desconto (R$ e %)
├── exercício_3/      # Verificador de Paridade e Sinal
├── exercício_4/      # Simulador de Investimento (Juros Simples + Bônus Compostos)
└── venv/             # Ambiente Virtual Python
